BSI 63 Auto-Sync

Fetches FFA Curves from SharePoint → Pushes BSI 63 data to GitHub

This page must be opened while logged into SharePoint (same browser session). The GitHub token is stored locally in your browser and never sent anywhere except GitHub's API.

Setup: Create a fine-grained PAT at github.com/settings/tokens with "Contents: Read and Write" permission for the smx-calculator repo.
